##task-manager

This is the simple application to manage your tasks.

###Usage

####Clone repository
```
git clone https://github.com/wlad031/task-manager.git task-manager
cd task-manager/
```

####Install & run
```
mvn clean install
java -jar target/taskmanager-1.0-SNAPSHOT.jar
```

Type 'help' to get the help message.
